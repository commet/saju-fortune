# 사주 데이터 정리

## 전체 요약
- **총 그룹**: 12개 가족/커플
- **총 인원**: 41명
- **역법 미상**: 18명 (확인 필요)
- **시간 없음/불명확**: 7명

## 데이터 주의사항
- `calendar: "미상"` → 원본에 양력/음력 명시 안 됨 (의뢰인 확인 필요)
- `birthTime: null` + `birthTimeRange` → 시간대로만 제공됨 (시주 결정 시 주의)
- `birthTime: null` + `birthTimeNote: null` → 출생시간 완전 미상
- `gender: null` → 이름만으로 성별 판단 불가 (박채운, 박채든, 황연우)
